# Future Features

Here are some features that are not currently in the PWA, but that you might want to add in the future as you enhance this app for your own portfolio.

  1. Ability to save entire search results.
  2. Pagination of results both on the server-side and in the app.
  3. Authentication and logging in to an account.
  4. Add user accounts so that the data will be tied to a single account. Users need to log in before they can access that data. Data stored on the device and can be seen, but data on the server needs to be connected to a user account.
  5. The data saved in the cache should be filtered and tied to a single user account too.
